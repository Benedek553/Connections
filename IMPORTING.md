## IMPORTING GOOGLE CONTACTS

### This guide explains how to import contacts from Google into Connections using a CSV export.

1) Export contacts from Google
- Open Google Contacts.
- Click Export.
- Choose "Google CSV".
- Save the file to your computer.

2) Start the import in Connections
- Open the table where you want to add contacts.
- Click "Import Google Contacts (CSV)".
- Select your exported CSV file.

3) Map CSV columns to Connections fields
- A mapping dialog appears with dropdowns for each field.
- Pick the column that should fill each field.
- Name uses two optional columns (First/Primary and Last/Secondary).
  Choose one or both; the values will be combined.
- You must select at least one Name column to continue.
- Unused fields can be left empty.

4) Finish
- Click "Import".
- A summary message shows how many contacts were imported and how many were skipped.
- Rows without a name are skipped.

Tips
- If your file does not have headers, re-export from Google Contacts.
- If you want to change the mapping, run the import again; it always asks for mapping.
